・am.py はKeithleyの現在地をambientに送るプログラムです。・ambient.py は私触ったことないです。とりあえず送ります。・DTAmain.py はDTA測定器を動かすメインのプログラムです。・Q1No.pyはDTAmainの中の質問事項でnと答えた場合に発動するプログラムです。・Q1Yes.pyはDTAmainの中の質問事項でyと答えた場合に発動するプログラムです。・Chset.pyはchinoに設定温度をセットするプログラムです。10行目の'/dev/ttyUSB2'のシリアルファイル名を適宜変更してください。11行目のボーレートも適宜変更してください。・Keigetpv.py はkeithleyの現在値を取得するプログラムです。中身はkeithley2000用と・keithley2182A用の関数があります。ほぼ同じです。・setScanrate.pyは開始温度、終了温度、昇温速度を設定するプログラムです。・vttotemp.pyは 取得した起電力データを温度に変換するプログラムです。・setSerKei.py はkeithley2000とkeithley2182Aにシリアル通信を行うプログラムです。5,6,16,17は適宜変更してくさい。
